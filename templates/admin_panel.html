Sesión Mesa Virtual

Si el servidor muestra "Error: No se completó el login a tiempo", la sesión de Mesa Virtual venció. Seguí estos pasos para renovarla:

  1. En tu computadora, pedile a Claude que ejecute mesa_virtual_iniciar_sesion
  2. Completá el login en la ventana de Chrome que se abre
  3. El archivo ~/.mesa_virtual_sesion.pkl se actualiza automáticamente
  4. Buscá ese archivo en tu computadora y subilo acá abajo
